# Hospital_SQL_Project(Query-based problem solving)
SQL project for hospital data analysis including appointments, billing, and treatments.

## Tools
- MySQL / SQL Workbench

## Problem Statement
Analyze hospital data to identify top doctors, highest paying patients, monthly trends, and treatment costs.

## Skills & Topics Covered

- SQL Basics: SELECT, WHERE, ORDER BY
- Aggregations: COUNT, SUM, AVG
- Grouping & Filtering: GROUP BY, HAVING
- Joins: INNER JOIN, multi-table joins
- Date Functions: YEAR(), MONTH(), MONTHNAME()
- Window Functions: RANK()
- Subqueries & Nested Queries

# Queries

List all patients’ first name, last name, and gender.
Retrieve all doctors who specialize in “Pediatrics”.
Count the total number of appointments for each doctor.
List all appointments scheduled for a specific date (e.g., today).
Show the total billing amount for each patient.
Find the total number of treatments per treatment type and show only types with more than 40 treatments.
List each doctor’s name along with the number of appointments they have had in January.
Retrieve patients who have pending payments along with their bill amount and treatment type.
Find the top 3 most expensive treatments.
Show the number of appointments per month for January, February, and March of the current year.
List the top 5 doctors with the highest number of completed appointments.
Find patients who have never had a treatment costing more than 50,000.
Show the total billing amount generated by each doctor by joining doctors, appointments, treatments, and billing tables.
Rank patients by the total amount they have paid and show the top 10 highest paying patients.
Calculate the average treatment cost per month for the current year and display trends month-wise.
